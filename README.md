# 🚀 DataSoc Technology Portfolio - Model Explorer App

An interactive web application built to visualize Machine Learning models and datasets. This project serves as a prototype for a **Model Automator** tool, aligning with the goals of the DataSoc Technology subcommittee.

## 😀 Link to website
https://datasoc-model-explorer-hp9ay8prglnrhvrdtntwtz.streamlit.app/#prediction-results-random-forest

## ✨ Key Features
- **Interactive Model Tuning:** Adjust hyperparameters (e.g., Number of Trees for Random Forest or K for KNN) and see accuracy updates in real-time.
- **Dynamic Visualization:** Live scatter plots generated from the Iris dataset.
- **Data Exploration:** Instant preview of raw data before model training.

## 🛠️ Tech Stack
- **Language:** Python
- **Framework:** Streamlit (Web UI)
- **Data Science:** Scikit-learn, Pandas
- **Visualization:** Seaborn, Matplotlib

## 🏃 How to Run Locally
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Launch the app: `streamlit run app.py`
